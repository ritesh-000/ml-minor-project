# ml-minor-project

## 📌 Project 1: Loan Approval Prediction System

### 🔍 Description:
This project predicts whether a loan application will be approved or not based on applicant details using machine learning algorithms.
It predicts loan approval based on applicant details like income, credit score, and employment status. We performed data cleaning, handled missing values, encoded categorical data, and scaled features. Then we trained models like Logistic Regression, KNN, and Naive Bayes and evaluated their performance using accuracy, precision, recall, and F1 score."

### 🛠 Technologies Used:
- Python
- Pandas
- NumPy
- Scikit-learn

### ⚙️ Process:
- Data preprocessing
- Handling missing values
- Feature selection
- Model training
- Prediction

  ## 📌 Project 2: Smart Card Customer Clustering

### 🔍 Description:
This project groups customers into different clusters based on their behavior using clustering techniques.
We used machine learning clustering techniques to segment customers based on their income, spending, and behavior. First we cleaned and transformed the data, then applied PCA for dimensionality reduction. After that, we used K-Means and Agglomerative clustering to group customers and analyzed each cluster to understand customer behavior.

### 🛠 Technologies Used:
- Python
- Pandas
- Matplotlib
- Scikit-learn

### ⚙️ Process:
- Data preprocessing
- Feature scaling
- Applying clustering algorithm (K-Means)
- Visualizing clusters

## 📊 Dataset Used:
